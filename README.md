# AI 2002 – Assignment 1: Uninformed Search Pathfinder

## Description
An interactive grid-based pathfinder that visualizes BFS, DFS, UCS, DLS, IDDFS,
and Bidirectional BFS with dynamic obstacle support.

## Requirements
- Python 3.x (no external packages needed — uses built-in Tkinter)

## How to Run
bash
python pathfinder.py


## Features
- 6 uninformed search algorithms
- Dynamic obstacles that spawn during search
- Step-by-step animation with adjustable speed
- Color-coded frontier, explored, and path nodes
